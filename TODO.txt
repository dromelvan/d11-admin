Use cases:

1. Update match day kickoff times.
    1a. Fetch current/upcoming match day.
    1b. Fetch all matches.
    1c. For each match, download match file.
    1d. Parse match file, read kickoff time.
    1e. Update match kickoff time.

2. Activate match day.
    2a. Fetch current match day.
    2b. If match day date >= today, activate match day.

3. Insert new player.
    3a. Download player file.
    3b. Parse player file, read player data.
    3c. Insert new player.
    
4. Update match stats.
    4a. Fetch matches for current date.
    4b. For each match, download match file.
    4c. Parse match file, read match stats.
    4d. Upload match xml file.
        4da. For each missing player, insert new player.
        4db. Repeat until no missing players.

5a. Finalize match stats.
    5a. Fetch matches for yesterday.
    5b. For each match, download match file.
    5c. Parse match file, read match stats.
    5d. Upload match xml file.
    5e. Finish match day.

5b. Finish match day.
    5a. Fetch current match day.
    5b. Fetch all matches.
    5c. If all match dates are < today - 1, finish match day.
